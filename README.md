# Solo-Fit-System
App gamificado estilo RPG inspirado em Solo Leveling que transforma hábitos e treinos em evolução de personagem (XP, atributos, equipamentos e ranks). Offline-first e pronto para Android.
⚔️ Solo Fit System

Transforme sua vida em um RPG. Evolua na vida real.

Solo Fit System é um aplicativo gamificado inspirado em Solo Leveling que transforma tarefas diárias, treinos e disciplina pessoal em um sistema completo de progressão estilo RPG.

Você não marca tarefas.
Você ganha XP.
Você não faz exercícios.
Você evolui atributos.
Você não completa a semana.
Você sobe de Rank.

🚀 Conceito

O Solo Fit System converte hábitos reais em mecânicas de jogo:

🎯 Missões diárias (academia, alimentação, tarefas customizadas)

📈 Sistema de XP e Level progressivo

🧬 Evolução de atributos (STR, DEX, CON, INT, CHA)

🛡️ Equipamentos com bônus passivos

💰 Moedas e loja interna

🏆 Sistema de Rank (E → S)

📊 Radar visual animado de atributos

🔥 Bônus semanal por consistência

Disciplina deixa de ser obrigação e vira progressão de personagem.

🧠 Filosofia do Projeto

O sistema é baseado em três pilares:

Gamificação realista – recompensas proporcionais ao esforço

Offline-first – funciona totalmente local (IndexedDB)

Progressão significativa – atributos impactam diretamente nos ganhos

Não é apenas um checklist com tema gamer.
É um sistema de evolução pessoal estruturado.

🛠️ Stack

HTML5 + CSS moderno (UI estilo sistema RPG)

JavaScript modular

IndexedDB (armazenamento local)

SVG animado para radar de atributos

Capacitor para build Android (APK)

📱 Android Ready

O projeto pode ser empacotado como APK via Capacitor.

Funciona offline e não depende de backend externo.
